## 属性
| 参数       | 说明  | 类型                     | 是否必填 | 默认值  |
|----------|-----|------------------------|------|------|
| name     | 名称  | string                 | 是    | `''` |
| fontSize | 大小  | number &verbar; string | 否    | -    |
| color    | 颜色  | string                 | 否    | -    |

## CSS 变量
| 属性               | 说明  | 默认值                         |
|------------------|-----|-----------------------------|
| --adt-icon-size  | 大小  | `var(--adt-font-size-main)` |
| --adt-icon-color | 颜色  | `var(--adt-color-text)`     |
